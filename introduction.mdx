ChatsPeCharcha is a modern real-time chat application that lets you connect with friends, exchange messages instantly, and stay aware of who's online. Built on React, Node.js, Socket.io, and MongoDB, it delivers a seamless messaging experience with features like typing indicators, read receipts, and a responsive dark/light mode UI.

## Get started in 3 steps

<Steps>
  <Step title="Clone and install">
    Clone the repository and install dependencies for both the backend and frontend.
  </Step>
  <Step title="Configure your environment">
    Set up your MongoDB connection, JWT secret, and optional Cloudinary credentials in `.env` files.
  </Step>
  <Step title="Start the servers">
    Run both the backend API server and the frontend Vite dev server, then open your browser.
  </Step>
</Steps>
