This guide walks you through cloning the ChatsPeCharcha repository, installing dependencies for the backend and frontend, configuring your environment variables, and starting both development servers so you can begin using the app locally.

<Note>
  ChatsPeCharcha requires **Node.js v18 or higher** and a running **MongoDB** instance (local or cloud). Make sure both are available before you begin.
</Note>

<Steps>
  <Step title="Clone the repository">
    Clone ChatsPeCharcha from GitHub and enter the project directory.

    ```bash
    git clone https://github.com/jatinpanday/chatsPeCharcha.git
    cd chatsPeCharcha
    ```
  </Step>

  <Step title="Install backend dependencies">
    Navigate to the `backend/` directory and install the Node.js dependencies.

    <CodeGroup>

    ```bash npm
    cd backend
    npm install
    ```

    ```bash yarn
    cd backend
    yarn install
    ```

    </CodeGroup>
  </Step>

  <Step title="Configure the backend environment">
    Copy the example environment file and fill in your values.

    ```bash
    cp .env.example .env
    ```

    Open `.env` and set the following variables:

    ```bash backend/.env
    PORT=5001
    MONGODB_URI=mongodb://localhost:27017/chatspecharcha
    JWT_SECRET=your_jwt_secret_here

    FRONTEND_URL=http://localhost:8080

    CLOUDINARY_CLOUD_NAME=your_cloud_name
    CLOUDINARY_API_KEY=your_api_key
    CLOUDINARY_API_SECRET=your_api_secret

    EMAIL_USER=your_email@example.com
    EMAIL_PASS=your_email_password
    ```

    See the [Configuration](/configuration) page for a full description of each variable.
  </Step>

  <Step title="Install frontend dependencies">
    Open a new terminal, navigate to the `frontend/` directory, and install dependencies.

    <CodeGroup>

    ```bash npm
    cd frontend
    npm install
    ```

    ```bash yarn
    cd frontend
    yarn install
    ```

    </CodeGroup>
  </Step>

  <Step title="Configure the frontend environment">
    Copy the example environment file and set the backend API URL.

    ```bash
    cp .env.example .env
    ```

    ```bash frontend/.env
    VITE_API_BASE_URL=http://localhost:5001
    ```
  </Step>

  <Step title="Start the backend server">
    From the `backend/` directory, start the development server with hot reload via Nodemon.

    <CodeGroup>

    ```bash npm
    npm run dev
    ```

    ```bash yarn
    yarn dev
    ```

    </CodeGroup>

    The backend will start on `http://localhost:5001` (or the `PORT` you configured).
  </Step>

  <Step title="Start the frontend server">
    From the `frontend/` directory, start the Vite development server.

    <CodeGroup>

    ```bash npm
    npm run dev
    ```

    ```bash yarn
    yarn dev
    ```

    </CodeGroup>
  </Step>

  <Step title="Open the app">
    Open your browser and navigate to the frontend dev server:

    ```
    http://localhost:8080
    ```

    You should see the ChatsPeCharcha login screen. Sign up for an account to get started.
  </Step>
</Steps>

<Tip>
  Run the backend and frontend in separate terminal windows so you can view logs from each server independently.
</Tip>
